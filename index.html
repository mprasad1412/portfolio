Recent Projects

MINI IT Project

MMU

This project was developed as my final semester project for the Foundation in Information Technology at Multimedia University (MMU), Malaysia. The website, built using HTML, CSS, and JavaScript, provides an interactive e-learning platform for children. It offers four courses that users can choose from, each accompanied by educational videos. After completing a course, children can test their knowledge by taking a quiz and receive an e-certificate upon successful completion. The website is designed to be engaging and user-friendly, providing a seamless learning experience for young users.

HUSTLE.INC Fitness App

UNITEN

This project focused on designing a user-friendly interface for a fitness app, developed using Figma. The app allows users to calculate their calorie intake and track calories burned through various workout programs. The goal was to improve the user experience by addressing common issues faced in existing apps like MyFitnessPal. The interface was designed to be intuitive, easy to navigate, ensuring a seamless experience for users to manage their fitness goals effectively.

RUNNING EVENT Website

UNITEN

This project was developed for the Web Programming subject and involved both frontend and backend development. The website was built using HTML, CSS, JavaScript, PHP, and SQL. It features two main user roles: admins and participants. Admins can manage categories and participants through CRUD operations, allowing them to create, read, update, and delete data. Participants can register for various event categories, view and edit their profiles, and track their participation.
