Recent Projects

RecyLink Mobile App

UNITEN (Final Year Project)

RecyLink is a comprehensive smart recycling application developed as my Final Year Project using Flutter and Firebase. The app features an AI-powered waste scanner powered by TensorFlow Lite, which allows users to instantly classify recyclable materials. It also includes a community marketplace for buying and selling upcycled goods and a gamified rewards system to incentivize sustainable habits. The project addresses the gap between waste management and circular economy participation through an intuitive mobile experience.

Serverless Facial Recognition

UNITEN (Cloud Architecture)

This project implements a secure, serverless access control system using AWS cloud infrastructure. Leveraging services such as AWS Lambda, Amazon Rekognition, DynamoDB, and S3, the system performs real-time identity verification without managing physical servers. Users upload images via a secure web interface, which are then processed by AI models to grant or deny access based on a confidence threshold. All access attempts are automatically logged for security auditing, demonstrating a cost-efficient and scalable cloud-native architecture.

ECE Car Rental System

UNITEN (Advanced Web Dev)

Easy Car Enterprise (ECE) is a robust online booking platform developed to digitize a multi-branch car rental business. Built with the Laravel PHP framework and Bootstrap, the system features complex logic including branch-specific fleet management, strict booking validation (overlap prevention), and role-based access for customers and staff. It streamlines the rental process by allowing users to filter vehicles by location and type while ensuring efficient inventory control for the business.

MINI IT Project

MMU

This project was developed as my final semester project for the Foundation in Information Technology at Multimedia University (MMU), Malaysia. The website, built using HTML, CSS, and JavaScript, provides an interactive e-learning platform for children. It offers four courses that users can choose from, each accompanied by educational videos. After completing a course, children can test their knowledge by taking a quiz and receive an e-certificate upon successful completion. RUNNING EVENT Website

UNITEN

This project was developed for the Web Programming subject and involved both frontend and backend development. The website was built using HTML, CSS, JavaScript, PHP, and SQL. It features two main user roles: admins and participants. Admins can manage categories and participants through CRUD operations, allowing them to create, read, update, and delete data. Participants can register for various event categories, view and edit their profiles, and track their participation.
